1. write a program to delete vowels from a given string using StringBuffer class.
2. write a program to create a class named Bank with following data members:
	name of depositor
	address of depositor
	account number
	balance in account
Class Bank has some methods for each of the following:
	generate unique account number for each depositor.
	display information and balance of depositor.
	deposit more amount in balance of any depositor.
	withdraw some amount from balance deposited.
	change the address of depositor.
After creating a class, do following operations:
	enter the information (name, address, account number and balance) of depositor. Number of depositor is to be entered by user.
	print the information of any depositor.
	add some amount to the account of any depositor and then display final information of that depositor.
	remove some amount from the account of any depositor and then display final information of that depositor.
	change the address of any depositor and then display the final information of that depositor.
	randomly repeat these processes for some bank accounts.
